# Lesson 03 — EF Core CRUD (Basic)

**Level:** Basic

## Goal

DbContext + DbSet<T>, SQLite setup, code-first migrations, basic CRUD (Add, Find, Update, Remove, SaveChangesAsync).

## Instructions

1. Read through the code changes in this branch (use `git diff` against the previous part).
2. Run the project and test the endpoints via the OpenAPI UI (`/openapi`).
3. Experiment — break things, fix them, extend them.
4. When ready, move to the next part branch.
